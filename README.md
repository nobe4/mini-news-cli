# `mini-news-cli` :newspaper:

Small CLI to https://www.newsminimalist.com/.

## Key mapping

| Key         | function                        |
| ---         | ---                             |
| `<UP>`      | up the list                     |
| `<DOWN>`    | down the list                   |
| `o`         | open the source in your browser |
| `q`/`<C-c>` | quit                            |
